This page summarizes field deployments and monitoring records connected to my research on structural monitoring, sensing systems, and infrastructure data.

Overview
======

- **50+ field experiments and deployments** across bridge monitoring, vehicle load sensing, precast transportation, port structures, sensor installation, DAQ validation, and cloud-based data review.
- **Four main field domains:** bridges, vehicles, precast transportation, and port structures.
- **Build-to-data workflow:** hardware, installation, DAQ, cloud upload, dashboard visualization, signal processing, and engineering interpretation.

Bridge Monitoring
======

- **Smart bridge IoT monitoring system**, 2021-2024
  Stand-alone sensing units, cloud upload, automated FFT/displacement analysis, and dashboard visualization for bridge health monitoring.

- **Field installation and response data**
  Sensor installation, response-history review, field validation, and troubleshooting under real bridge conditions.

- **Field sites and applications**
  Paldang Overpass, Cheongdam Bridge, Seongsan Bridge, Seosomun Overpass, and other bridge inspection/diagnosis sites in Korea.

- **Measured data**
  Strain, acceleration, displacement, temperature, vibration characteristics, communication status, and power reliability.

Transportation and Load Monitoring
======

- **Vehicle load sensing and WIM DAQ**, 2024-present
  High-speed signal acquisition, WIM DAQ validation, load interval detection, field driving data review, and vehicle-weight analysis workflows.

- **OBM field validation**
  Vehicle-state sensing, dynamic weight correction, and validation against WIM-related field conditions.

- **Precast transportation monitoring**, 2020-2023
  Portable sensing of precast panels and concrete girders during lifting, loading, driving, and installation.

- **Measured data**
  High-speed load signals, vehicle events, acceleration, strain, tilt, transport vibration, support-condition effects, and abnormal transport events.

Sensing and Deployment
======

- **Sensor installation and maintenance**
  Field setup of IoT sensors, JANET sensors, gateways, embedded strain gauges, DAQ devices, packaging, power, and communication modules.

- **Cloud-connected data workflows**
  Cloud databases, monitoring dashboards, chatbot upload, QR-based data acquisition, and AI-assisted report generation.

- **Ruggedized port monitoring**
  Waterproof sensor packaging, harsh-environment operation, event triggering, and multisensory event records for port and quay wall monitoring.

What Each Field Case Records
======

- **Date:** year, month, or project period for the field campaign.
- **Site:** bridge, road, precast yard, port, lab-to-field validation site, or anonymized site name.
- **Role:** sensor installation, DAQ setup, driving test, cloud review, troubleshooting, or data analysis.
- **Hardware:** sensor, gateway, DAQ, power, communication, and packaging details.
- **Data:** signal type, sampling rate, event type, dashboard, visualization, or analysis output.
- **Evidence:** photo, dashboard screenshot, signal visualization, project slide, or publication/patent link.
