# Field Sites

This page summarizes real-world sites where I have deployed, operated, or reviewed sensing and monitoring systems. I keep this page separate from Research because it is more about field context, measured data, and deployment experience than project categories.

## Bridge and Infrastructure Sites

- **Seongsan Bridge**
  Bridge monitoring and field data review using sensing systems for structural response measurement.
  Data handled: strain, acceleration, vibration characteristics, displacement-related estimation records, cloud upload status, and monitoring logs.

- **Seosomun Overpass**
  Field deployment and monitoring experience for bridge response measurement under real operating conditions.
  Data handled: strain, acceleration, temperature, vibration response, sensor communication status, and system operation records.

- **Paldang Bridge / Paldang Overpass**
  Field monitoring and bridge inspection-related measurement experience.
  Data handled: structural response histories, vibration characteristics, field installation records, and monitoring-system reliability data.

- **Cheongdam Bridge and Other Bridge Sites**
  Bridge inspection, diagnosis, and monitoring support across multiple field campaigns in Korea.
  Data handled: sensor installation records, response histories, environmental conditions, and engineering interpretation notes.

## Transportation and Vehicle Monitoring Sites

- **WIM Field Tests**
  High-speed vehicle load sensing and WIM DAQ validation under driving conditions.
  Data handled: high-speed load signals, load-event intervals, filtered/downsampled signals, axle-related records, and event labels.

- **OBM Field Validation**
  Vehicle-mounted sensing for dynamic operating-weight estimation.
  Data handled: vehicle-state signals, dynamic weight correction data, validation records, and comparison data against WIM-related conditions.

- **Freight and Rail-related Monitoring**
  Event-oriented field sensing and signal review for transportation safety monitoring.
  Data handled: event records, time histories, field measurement logs, and signal review notes.

## Prefabricated and Port Structure Sites

- **Precast Transportation Monitoring**
  Portable sensing for precast members during lifting, loading, transportation, and erection.
  Data handled: strain, acceleration, movement-stage records, transportation events, and installation-stage response histories.

- **Port / Quay Wall Monitoring**
  Ruggedized multisensory monitoring for quay wall and port-structure events.
  Data handled: event-triggered sensor records, waterproof sensor operation logs, cloud communication data, and AI-assisted event labels.
