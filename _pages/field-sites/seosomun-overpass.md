## System Configuration

  <img src="/images/field-sites/seosomun-overpass/system-overview.jpg" alt="Seosomun Overpass monitoring system installed below the bridge deck">

- Concrete overpass monitoring site located around Jungnim-dong and Sunhwa-dong, Jung-gu, Seoul.
- Field system configured for three-axis acceleration, three-channel strain, and one-channel temperature measurement.
- Measurements were organized as 100 Hz records collected in 10-minute windows every hour.
- This entry focuses on field deployment and system configuration rather than detailed data-analysis results.
