Transportation Monitoring Systems
======

- **High-Speed WIM System Development**  
  Load segment detection, high-speed vehicle sensing, and dynamic truck-weight correction workflows.

- **OBM (On-Board Mass) System Development**  
  Vehicle-mounted load monitoring and sensor-based mass estimation.

- **Freight Train Monitoring System**  
  Field sensing workflows for freight and rail-related monitoring.

- **Prefabricated Structure Transportation Monitoring System**  
  Portable sensing for transportation-stage monitoring of precast members.

Structural Monitoring Systems
======

- **Cloud-Based Long-Term Bridge Monitoring System**  
  Long-term bridge response monitoring using IoT sensors and cloud dashboards.

- **Bridge Load-Carrying Capacity Evaluation System**  
  Field data workflows for bridge load testing and performance evaluation.

- **QR-Based Bridge Maintenance Monitoring System**  
  QR-linked lifecycle and maintenance monitoring records.

- **Port Structure Monitoring System**  
  Multisensory monitoring for quay wall and port structure events.

Smart Sensing Systems
======

- **Smart Concrete Sensing System**  
  Concrete crack and trajectory monitoring using sensing data.

- **Seismic Sensing System**  
  Low-power sensing for seismic response monitoring.
