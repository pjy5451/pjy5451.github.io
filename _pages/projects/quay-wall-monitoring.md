# Port Structure Monitoring System

Port structure monitoring work focused on multisensory event monitoring for quay wall and waterfront infrastructure.

## Problem

Port structures operate in harsh environments where salt exposure, water, impact events, and communication instability can make continuous monitoring difficult.

## My Role

- Supported ruggedized sensing workflows for waterfront infrastructure monitoring.
- Reviewed event-triggered field data and cloud communication records.
- Connected multisensory data to AI-assisted event recognition.

## System / Methods

- Waterproof sensor packaging and solar-assisted operation.
- LTE/cloud communication and watchdog-based fault-tolerant operation.
- Multisensory monitoring of vibration, tilt, and event conditions.
- AI-assisted event recognition for quay wall data.

## Field Deployment

- Applied monitoring concepts to port and quay wall conditions, including Incheon Port-related field context.
- Reviewed field readiness under waterproofing, power, and communication constraints.

## Data / Outputs

- Event-triggered sensor records, vibration/tilt histories, and cloud communication logs.
- Operation records for ruggedized monitoring in harsh environments.
- AI-assisted labels and event interpretation workflows.

## Related Publications / Outputs

- Development of AI-Based Multisensory System for Monitoring Quay Wall Events.
- Development of AI-Cloud Based High-Sensitivity Wireless Smart Sensor for Port Structure Monitoring.
