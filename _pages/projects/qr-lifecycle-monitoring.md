# QR-Based Bridge Maintenance Monitoring System

QR-based monitoring connects field maintenance records with low-power sensing, visualization, and cloud reporting.

## Problem

Long-term maintenance data can be difficult to collect when sensors lack communication modules or when field inspection workflows require separate applications and manual reporting.

## My Role

- Developed QR-linked monitoring concepts for low-power infrastructure sensing.
- Designed data collection workflows using QR visualization, chatbot upload, and cloud storage.
- Connected measured data to automated reporting concepts.

## System / Methods

- Ultra-low-power strain sensing with e-paper QR visualization.
- QR decoding and cloud database workflows.
- Chatbot-assisted data upload without a dedicated mobile application.
- AI-assisted report generation from collected sensor data.

## Field Deployment

- Applied QR-based sensing concepts to bridge lifecycle and maintenance monitoring scenarios.
- Installed embedded strain and temperature sensing in a field bridge construction context.

## Data / Outputs

- Static strain, temperature, QR-encoded data, and inspection records.
- Cloud-stored maintenance histories.
- Data-driven report outputs for field maintenance review.
