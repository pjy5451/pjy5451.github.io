# Sewer Monitoring Sensor System

This smart sensing project focused on a field-oriented sensor concept for sewer and underground infrastructure monitoring.

## Problem

Underground and sewer infrastructure monitoring requires sensing systems that can operate in constrained environments while collecting data useful for smart-city maintenance decisions.

## My Role

- Developed early-stage smart sensing concepts for underground infrastructure monitoring.
- Reviewed field-oriented packaging, operation, and data collection requirements.
- Connected sensor concepts to smart-city and environmental monitoring applications.

## System / Methods

- Field-oriented sensor concept development for sewer monitoring.
- Low-power IoT sensing and data collection workflow design.
- Environmental and infrastructure monitoring scenario review.
- Smart-city maintenance workflow concept development.

## Field Deployment

- Treated as an early-stage sensing concept for sewer and underground infrastructure conditions.
- Used lessons from smart-city sensing and low-power monitoring projects.

## Data / Outputs

- Sensor concept records and monitoring scenario definitions.
- Smart-city sensing workflow ideas for sewer and underground infrastructure.
- Award/project materials related to IoT and deep-learning-based smart water grid concepts.

## Related Publications / Outputs

- IoT and deep-learning-based smart water grid concept for sewer and water-pipe monitoring.
- Smart-city sensing workflow materials for underground infrastructure monitoring.
