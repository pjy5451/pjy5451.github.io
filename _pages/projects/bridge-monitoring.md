# Cloud-Based Long-Term Bridge Monitoring System

<div class="quick-links">
  <a href="/projects/">Back to Projects</a>
  <a href="/publications/">Related Publications</a>
</div>

This project developed long-term bridge monitoring workflows using IoT sensors, cloud infrastructure, and engineering interpretation of field data.

## Focus

- Stand-alone IoT strain and acceleration sensing.
- Cloud upload, database storage, and dashboard visualization.
- FFT analysis and vibration-characteristic review.
- Strain-acceleration data fusion for bridge displacement estimation.
- Field deployment on actual bridge structures in Korea.
