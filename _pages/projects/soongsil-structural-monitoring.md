# Soongsil University Structural Monitoring System

This project covers structural monitoring application and sensing-data review for a university infrastructure site.

## Focus

- Field-oriented structural response monitoring.
- Sensor operation and measurement-log review.
- Data organization for structural monitoring interpretation.
- Deployment-context documentation for future project records.
