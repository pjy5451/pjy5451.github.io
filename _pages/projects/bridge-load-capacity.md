# Bridge Load-Carrying Capacity Evaluation System

This work supports bridge load testing and load-carrying capacity evaluation through field sensing and response interpretation.

## Problem

Bridge load-carrying capacity evaluation requires reliable field measurements, but conventional load tests can be limited by sensor installation constraints and short measurement windows.

## My Role

- Planned sensing workflows for bridge load tests and field response measurement.
- Reviewed measured response histories and organized load-related records.
- Connected portable multimetric sensing data to structural interpretation.

## System / Methods

- Portable sensing for load-test response measurement.
- Acceleration, strain, and displacement-related response review.
- Vibration-characteristic analysis and field-data organization.
- Event-based interpretation for load-carrying capacity evaluation.

## Field Deployment

- Supported field measurement workflows for bridge load testing and load-carrying capacity evaluation.
- Used practical installation and response-review experience from bridge monitoring projects.

## Data / Outputs

- Load-test response histories and vibration characteristics.
- Field measurement records for structural evaluation workflows.
- Multimetric sensing evidence for load-carrying capacity assessment.

## Related Publications / Outputs

- Development of a Portable Multimetric Sensing System for Bridge Load Testing and Load-Carrying Capacity Evaluation.
- Automatic Load-Event Interval Recognition Using IoT Sensors for Structural Capacity Assessment.
