Selected Project Experience
======

- **Bridge Monitoring and Displacement Estimation**, 2021-2024
  Developed and deployed IoT sensor and cloud-based bridge monitoring workflows. The work connects synchronized strain/acceleration sensing, cloud upload, database storage, dashboard visualization, FFT analysis, and displacement estimation. Field deployments include Paldang Overpass, Cheongdam Bridge, Seongsan Bridge, and other bridge monitoring sites in Korea.

- **High-Speed WIM System Development**, 2025-present
  Developed analysis workflows for high-speed weigh-in-motion signals, including DAQ validation, filtering, downsampling, event labeling, and YOLO-based automatic load segment detection.

- **OBM (On-Board Mass) System Development**, 2024-present
  Worked on vehicle-state sensing and dynamic operating-weight correction using field-ready monitoring and FIR-filter-based signal correction workflows.

- **Portable Sensing for Precast Transportation**, 2020-2023
  Developed a portable JANET sensing workflow for monitoring precast members during lifting, loading, transportation, and installation. The system measured acceleration, strain, tilt, transport vibration, and support-condition effects.

- **AI-Based Quay Wall Event Monitoring**, 2021-2023
  Worked on ruggedized multisensory monitoring for port and quay wall events, including waterproof packaging, cloud communication, event triggering, and AI-assisted event recognition.

- **Smart IoT Earthquake Sensor**, 2019
  Developed a low-cost IoT earthquake sensing and notification concept using STA/LTA triggering, time-domain and frequency-domain classification, and BLE-based communication. The project received the Minister of the Interior and Safety Award.
