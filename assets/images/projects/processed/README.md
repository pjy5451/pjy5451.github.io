# Processed Project Images

Use this folder for website-ready project images.

Recommended workflow:

1. Put original files in `assets/images/projects/raw/`.
2. Crop out confidential headers, company names, faces, license plates, and client-sensitive details.
3. Export a clean web image into this folder.
4. Reference only processed images from project pages.

Suggested dimensions:

- Project card image: `1200 x 400` or similar wide crop.
- Project detail image: `1600 x 700` or `1600 x 900`.
- Field evidence image: `1600 x 800` or `1600 x 1000`.
