# Field Work Image Folder

Use this folder for cleaned images that will appear on the `Field Work` page.

Recommended file naming:

- `bridge-installation-YYYY-site-shortname.jpg`
- `wim-daq-YYYY-test-shortname.jpg`
- `precast-transport-YYYY-site-shortname.jpg`
- `port-monitoring-YYYY-site-shortname.jpg`

Image preparation checklist:

- Crop out confidential or distracting slide headers, including company names when needed.
- Remove personal information, license plates, site-sensitive labels, and client-sensitive details.
- Prefer landscape images around `1600 x 800` or `1600 x 1000`.
- Keep one clear subject per image: sensor installation, DAQ setup, dashboard, signal visualization, or field scene.
- Add a short caption in the page rather than embedding text inside the image.
