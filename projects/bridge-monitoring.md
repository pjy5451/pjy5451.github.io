## Overview

This project focuses on scalable bridge structural health monitoring using low-power IoT sensing, cloud-based data handling, and displacement estimation. It formed the core direction of my master's thesis: *A scalable Bridge Health Monitoring System using an IoT sensor and Cloud computing*.

## My Role

I worked on the monitoring system concept, IoT sensing workflow, cloud-based data handling, data fusion logic, and research documentation. The project connected field measurement constraints with a scalable data pipeline for long-term bridge monitoring.

## Methods

- Low-power IoT sensing
- Strain and acceleration measurement
- Cloud-based data fusion
- Reference-free displacement estimation

## Results

The work produced journal publications, preprints, and patent records related to bridge monitoring, IoT sensing, and cloud-based data preprocessing.
